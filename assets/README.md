Place your token logo here.

Requirements:
- Square (1:1 aspect ratio) — 512×512 or 1024×1024 recommended
- PNG, JPG, WebP, or GIF
- Transparent background works well on dark and light UIs

Usage:
- Set BANKR_TOKEN_IMAGE=./assets/token-logo.png in .env
- Bootstrap resolves it to a raw GitHub URL automatically
- Or use any hosted https:// URL directly
